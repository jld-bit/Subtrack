# SubTrack

SubTrack is an original SwiftUI iOS subscription tracker implementation with local Core Data storage.

## Features
- Add subscriptions with name, price, billing date, and category icon.
- Review monthly spending at a glance.
- Browse subscription cards with next renewal dates.
- Receive local billing reminders before renewal.
- View category-based spending analytics chart (unlocked with IAP).

## Monetization
SubTrack uses StoreKit 2 with a non-consumable in-app purchase:
- Product ID: `com.subtrack.unlimited`
- Unlocks unlimited subscriptions and spending analytics.

## Notes
- This project intentionally uses an original implementation and does not copy branding, design assets, or visual identity from any existing subscription tracker app.
